# Picture Worth a Thousand Lessons Portal

Static portal page linking to:

- Version 1: https://kr48vr.github.io/Picture-Worth-Thousand-Lessons/
- Version 2: https://kr48vr.github.io/Picture-Worth-Thousand-Lessons-Upscale/

Upload the contents of this folder to a GitHub Pages repository. The portal intentionally uses newly generated handmade-style visuals rather than previewing the wildlife imagery, so the linked sites remain a surprise.

Files:

- `index.html`
- `portal-20260624c.css` — versioned stylesheet to avoid GitHub Pages cache mismatch
- `styles.css` — fallback copy of the stylesheet
- `assets/hero-hands-maker-v5.jpg` — pen-and-watercolor photography/making montage with matching camera/phone screens and better-anchored hands
- `assets/flower-science-hands.jpg` — watercolor flower/science image
- `assets/feedback-impasto-tech.jpg` — playful impasto oil-painting feedback-loop image with high-tech making
- `assets/craft-hands-charcoal-tech-v2.jpg` — black-and-white charcoal craft/high-tech making image with anchored hands and forearms
- `assets/version-1-site.jpg` and `assets/version-2-site.jpg` — distinct faint screenshot backgrounds for the two experiment cards

GoatCounter is included with site endpoint `https://kr48vr.goatcounter.com/count` via the standard `gc.zgo.at/count.js` script.
